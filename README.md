# Simple Responsive Landing Page

## Project Overview
This project is a simple, responsive landing page made using HTML and CSS. It includes a header, hero section, about section, and footer. The page adapts for both desktop and mobile screen sizes.

## Features
- Clean HTML5 layout with header, sections, and footer
- Styled using CSS, including Flexbox and media queries for responsiveness
- Includes navigation links and a call-to-action button
- Footer with social media links

## Files Included
- index.html — the main page structure
- style.css — the styling and responsive rules

## How to Run
1. Clone or download this repo to your computer.
2. Open index.html in your browser (recommend VS Code + Live Server).
3. View and test the page on desktop and mobile for responsive design.
